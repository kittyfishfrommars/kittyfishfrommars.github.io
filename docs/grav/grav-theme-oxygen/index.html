Look, it's Kittyfish from Mars.

Oxygen is a crazy fast, accessible and GDPR-compliant theme for Grav CMS.
Available for free for personal and commercial use - it's source code and build tools are available at github.com/kittyfishfrommars.
Provided as-is. No strings attached. Now with modular building blocks!

All-in-One Package

A Grav skeleton containing this theme, the core Grav system plus sample pages, plugins and configuration.
